# Party Bar Order System

## Setup & Run

### 1. Server (Backend)
Open a terminal:
```bash
cd server
npm start
```
Server runs on http://localhost:3001

### 2. Client (Frontend)
Open another terminal:
```bash
cd client
npm run dev
```
Open http://localhost:5173 in your browser.

## Features
- **User:** / (Root) - Order drinks.
- **Admin:** /admin - Manage ingredients, Create menu (Drag & Drop), View live orders.
